HTML

HTML is a hyper text markup language which is used to help in builds a webpage.HTML is used to define the structure of webpage and also there are so many language(CSS,Javascript,etc.) which helps in buils a webpage at an exordinary point.It is interaction between a text language into web browser.

It is not a programming language.

This Html was developed by WHATWG(Web Hypertext Application Technology Working Group).

Hypertext word means it is that text from which you can naviagte to another web page . For example click on hyper.you will get an by cliking on an hyper link.

Markup word means it is used to annotate text, images and other content in a web page.Markup includes the elements such as head, title, bod, header, footer, article, section, p, div, img, etc.

Html tags are used to mark for the opening and closing tags.

<!Doctype html> stands for Document type file.It informs to web browser which language is written in fileand it tells the browser to which version Html document is using .

<html> Sometimes,It is known as root element in the file and it wrap all the content of the entire page.
